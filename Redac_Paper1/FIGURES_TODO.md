## Figure Preparation Guidelines

### Technical Requirements (EES)
- **Resolution**: Minimum 600 dpi (1200 dpi preferred)
- **Format**: TIFF, EPS, or PDF (vector preferred)
- **Color**: RGB for online, CMYK for print
- **File size**: <10 MB per figure

### Best Practices
- Clear, readable axis labels (≥10 pt font)
- Color-blind friendly palettes
- Consistent styling across all figures
- Legends positioned inside plot area when possible
- Error bars on all data points

### Software Recommendations
- **Python**: matplotlib, seaborn (publication-quality)
- **MATLAB**: built-in plotting functions
- **Inkscape**: vector graphics editing
- **Adobe Illustrator**: professional layout
